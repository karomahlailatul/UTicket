# uTicket 

## About the project
uTicket Built up with NextJs with backend ExpressJS and deployment on Vercel.

## Usage 

### Clone
```shell
git clone https://github.com/karomahlailatul/uTicket.git
```
### Installing 
```shell
npm i;
```

### Testing
```shell
npm run start;
```
